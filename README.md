## Installation

1. Clone the repository in your local folder.
```
git clone https://github.com/Juakata/Battleship-JS.git
```
2. Run
```
npm install
```
```
npx webpack
```
3. Open index.html in your browser.

4. To test run
```
npm run test
```

5. To run the project in production mode (creates dist folder)
```
npm run build
```
